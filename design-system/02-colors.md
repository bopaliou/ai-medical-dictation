# Palette de Couleurs KadduCare

## Vue d'ensemble

La palette de couleurs KadduCare est extraite du logo officiel et reflète les valeurs de la marque : confiance médicale, modernité et bienveillance.

## Couleur Principale

### KadduCare Blue (#0A84FF)

La couleur principale de KadduCare est un bleu médical moderne qui inspire confiance et professionnalisme.

```
Primary: #0A84FF
RGB: rgb(10, 132, 255)
HSL: hsl(211, 100%, 52%)
```

**Usage** :
- Boutons principaux
- Liens actifs
- Éléments d'interaction
- Accents et highlights
- Icônes principales

**Variantes** :
- **Primary Light** : #E8F1FF (backgrounds, accents légers)
- **Primary Dark** : #0051D5 (contrastes, profondeur)

## Couleur Secondaire

### Health Green (#34C759)

Le vert santé représente les validations, les statuts positifs et les actions réussies.

```
Success: #34C759
RGB: rgb(52, 199, 89)
HSL: hsl(142, 61%, 49%)
```

**Usage** :
- Statuts "finalisé"
- Messages de succès
- Indicateurs positifs
- Badges de validation

**Variantes** :
- **Success Light** : #E8F5E9 (backgrounds)
- **Success Dark** : #28A745 (contrastes)

## Contraste et Accessibilité

Toutes les combinaisons de couleurs respectent les standards WCAG :

- **WCAG AA** : Contraste minimum 4.5:1 pour le texte normal
- **WCAG AAA** : Contraste minimum 7:1 pour le texte important

### Combinaisons Approuvées

#### Texte sur Fond Clair

- Texte Principal (#1B1B1D) sur Background (#F5F5F7) : ✅ 16.5:1
- Texte Principal sur Background Card (#FFFFFF) : ✅ 16.5:1
- Texte Secondaire (#4A4A4A) sur Background Card : ✅ 7.1:1

#### Texte sur Fond Sombre

- Texte Principal (#F2F2F3) sur Background (#0D0D0F) : ✅ 16.5:1
- Texte Principal sur Background Card (#161618) : ✅ 16.5:1

#### Couleurs d'Action

- Primary (#0A84FF) sur Blanc (#FFFFFF) : ✅ 4.5:1
- Success (#34C759) sur Blanc : ✅ 4.5:1
- Error (#FF3B30) sur Blanc : ✅ 4.5:1

## Usage par Contexte

### Interface Médicale

- **Bleu** : Actions principales, navigation
- **Vert** : Validations, statuts positifs
- **Rouge** : Alertes, erreurs, suppressions
- **Orange** : Avertissements, brouillons
- **Gris** : Textes, bordures, éléments neutres

### Hiérarchie Visuelle

1. **Couleur primaire** : Éléments les plus importants
2. **Couleur secondaire** : Éléments secondaires
3. **Neutres** : Fond et structure
4. **États** : Feedback et statuts
