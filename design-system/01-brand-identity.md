# Identité de Marque KadduCare

## Logo Principal

Le logo KadduCare est l'élément central de l'identité visuelle. Il représente la confiance, la modernité et l'innovation médicale.

### Logo Standard

**Fichier** : `logo-kadducare.png`  
**Emplacement** : `mobile/mobile-app/assets/images/logo-kadducare.png`

Le logo KadduCare est composé d'un symbole graphique combinant des éléments modernes et médicaux, accompagné du nom de la marque.

### Tailles Minimales

Pour garantir la lisibilité et l'impact visuel, le logo doit respecter les tailles minimales suivantes :

- **Mobile (écran)** : 60px minimum
- **Header d'application** : 70px recommandé
- **Splash Screen** : 120px pour l'impact maximum
- **Icône d'application** : 1024x1024px (iOS), 512x512px (Android)

### Clear Space

Le clear space (espace de respiration) est l'espace minimum qui doit entourer le logo pour préserver son impact visuel.

**Règle** : L'espace minimum autour du logo doit être égal à **50% de la hauteur du logo**.

```
┌─────────────────────────┐
│                         │
│    ┌───────────┐        │
│    │           │        │
│    │   LOGO    │        │ ← Clear space = 50% de la hauteur
│    │           │        │
│    └───────────┘        │
│                         │
└─────────────────────────┘
```

### Variantes du Logo

#### Version Standard (Clair)

Utilisée sur les fonds clairs et dans la majorité des contextes.

- **Fond recommandé** : Blanc (#FFFFFF) ou gris très clair (#F5F5F7)
- **Usage** : Headers, écrans principaux, documentation

#### Version Sombre

Pour les fonds sombres et le mode sombre de l'application.

- **Fond recommandé** : Noir (#0D0D0F) ou gris foncé (#1A1A1D)
- **Usage** : Mode sombre, écrans avec fond sombre

#### Version Monochrome

Version simplifiée pour les contextes où la couleur n'est pas disponible.

- **Couleur** : Noir (#1B1B1D) ou blanc selon le fond
- **Usage** : Impressions, documents monochromes

### Usage du Logo

#### ✅ Bonnes Pratiques

- Utiliser le logo avec le clear space respecté
- Maintenir les proportions originales
- Utiliser la version appropriée selon le fond
- Placer le logo de manière visible mais non intrusive

#### ❌ À Éviter

- Ne pas déformer ou étirer le logo
- Ne pas modifier les couleurs du logo
- Ne pas placer le logo sur des fonds qui nuisent à sa lisibilité
- Ne pas utiliser le logo trop petit (en dessous des tailles minimales)
- Ne pas ajouter d'effets (ombres, contours, dégradés) au logo

### Placement dans l'Application

#### Header Principal

Le logo apparaît dans le header de chaque écran principal :

- **Taille** : 70px
- **Position** : Gauche, aligné avec le texte "KadduCare"
- **Espacement** : 12px entre le logo et le texte

#### Splash Screen

Pour l'écran de démarrage :

- **Taille** : 120px
- **Position** : Centré verticalement et horizontalement
- **Animation** : Fade-in avec légère animation de scale

#### Icône d'Application

Pour les icônes système :

- **iOS** : 1024x1024px
- **Android** : 512x512px (adaptive icon)
- **Format** : PNG avec transparence

### Couleurs du Logo

Le logo KadduCare utilise principalement :

- **Bleu principal** : #0A84FF (KadduCare Blue)
- **Bleu foncé** : #0051D5 (pour les accents)
- **Vert santé** : #34C759 (pour les éléments médicaux)
- **Blanc** : #FFFFFF (pour les versions claires)
- **Noir** : #1B1B1D (pour les versions sombres)

Ces couleurs sont extraites du logo et forment la base de la palette de couleurs KadduCare.

### Personnalité Visuelle

Le logo KadduCare communique :

- **Modernité** : Design contemporain et épuré
- **Confiance** : Formes stables et équilibrées
- **Innovation** : Esthétique technologique
- **Humanité** : Courbes douces et approche bienveillante
- **Professionnalisme** : Précision et rigueur médicale
